# License-Pay

## Description
A project for [Meichuhackathon 2022](https://2022.meichuhackathon.org/)
Our goal is to create a digital payment method using car plates.

Three folders stands for three main parts of this project respectively
- License-Plate-Detector: to translate a carplate image into number
- license_web: The frontend, display the website for requesting and making payment.
- server: The backend, communicate with a psql database, and handles requests from frontend.
